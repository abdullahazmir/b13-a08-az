# 🐄 QurbaniHat – Livestock Booking Platform

## 🌐 Live Site
https://b13-a08-az.vercel.app/

---

## 📌 Project Purpose

QurbaniHat is a modern livestock marketplace where users can explore animals such as cows, goats, sheep, and buffalo for Qurbani. Users can view detailed information and place a booking after authentication.

---

## 🚀 Key Features

- 🏠 Responsive Home Page with Banner and Featured Animals
- 🐄 Browse All Animals with Price Sorting (Low → High)
- 🔍 Detailed Animal View 
- 🔐 Authentication System (Better Auth)
  - Email & Password Login
  - Google Login
- 👤 My Profile Page (User info display)
- ✏️ Update Profile Feature
- 🚫 Custom Not Found Page
- 🎨 Smooth UI with Animate.css

---
